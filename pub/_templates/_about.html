About the project

This project created as a learning and experimental playground. The main field I want to hone my skills with it are AngularJS, authorisation mechanics and user models.

Functionality

Current

  • Items catalogue
  • Catalogue filters
  • i18n (internationalization)
  • Cart
  • Checkout

Roadmap

  • Unit tests
  • Integrate Firebase
  • Login and authentication
  • User groups policies
  • Admin user group, members able to edit content
  • Integrate Filestack as a files CDN
  • Soft transitional animation
  • Images preloader
  • Pages preloader

Future goals

  • Migrate to own back-end
  • Upgrade to Angular 2.0

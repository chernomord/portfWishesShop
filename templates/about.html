About the project

Created it for fun and to practice with Angular. At the current state of development Angular is pretty well practiced and I feel the next step should be back-end integration, authentication mechanics implementation on both sides of the app and content editable for admin user group. In other words - less front-end, more business logic.

Tech used during development of the project

  • Angular
  • Bootstrap
  • angular-UI
  • ui-router
  • angular-translate
  • ng-tags-input
  • Gulp
  • Jade
  • Stylus

Current functionality

  • Main
    • Catalogue
    • Item details
    • Catalogue filters
    • Cart
    • Checkout
    • i18n, partial (lack of translation)
  • Technical
    • Data loaded via AJAX from JSON files
    • Responsive Cart button
    • Cart content stores in cookie
    • Cart item remove/clear confirmation dialogue
    • Cart item qty field input restricted only to integer
    • Img loading indicator
    • Sticky menus and filter
